# 본인인증 견적 비교 체크리스트

아래 항목을 동일 조건으로 요청하면 PG/대행 견적을 정확히 비교할 수 있습니다.

## 1) 사업자/서비스 기본 정보
- 사업자 유형: 개인사업자
- 업종/서비스 유형: (예: 콘텐츠 플랫폼/커뮤니티/구독)
- 서비스 런칭 시점: (예: 2025-03)
- 월 예상 인증 건수: 5,000건
- 월 결제 거래액(예상): (예: 3,000만원)

## 2) 본인인증 범위
- 인증 목적: 성인 인증 (만 19세 이상)
- 인증 수단: 휴대폰 본인인증(PASS 포함)
- 인증 실패 처리 정책: 재시도/차단/안내 문구
- 재인증 주기: (예: 최초 1회만)

## 3) 연동 방식
- PG 사용 여부: 토스페이먼츠 사용 예정
- 본인인증 연동 희망: 단독/PG와 묶음/대행(포트원 등)
- 기술 방식: API 연동 / 리다이렉트 / 팝업
- 콜백 방식: 서버 콜백 필요 여부

## 4) 비용 요청 항목
- 건당 과금 단가
- 월 정액 요금제 옵션
- 월 최소 이용료 여부
- 초기 가입비/세팅비
- 환불/취소 정책
- 초과 건 과금 방식

## 5) 운영/정책
- 성인 인증 로그 보관 기간
- 개인정보 처리/보관 방식
- SLA(장애 대응) 유무

## 6) 계약/심사
- 개인사업자 가능 여부
- 심사 소요 기간
- 필요 서류(사업자등록증/서비스 화면 캡처 등)

---

## 추천 비교 요청 문구 (복사해서 사용)
안녕하세요. 개인사업자이며 성인 인증(휴대폰/PASS) 도입을 검토 중입니다.
월 예상 인증 건수는 약 5,000건이고, 결제 PG는 토스페이먼츠 사용 예정입니다.
API 연동 기준으로 다음 조건의 견적을 부탁드립니다.

1) 건당 단가/월 정액 요금제 옵션
2) 월 최소 이용료 및 초기 가입비
3) 초과 건 과금 방식
4) 기술 연동 방식(API/리다이렉트/팝업)
5) 개인사업자 계약 가능 여부 및 심사 기간

감사합니다.
